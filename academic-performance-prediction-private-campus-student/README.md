# Academic Performance Prediction - Private Campus Students

This is our project repository for the IT41043 Intelligent Systems research assignment.

## Topic

A Comparative Study of Academic Performance Prediction Among Sri Lankan Private Campus Students Using Survey-Based Factors.

## What we are trying to do

We collected survey responses from private campus students in Sri Lanka.  
The plan is to use the survey answers to predict the student's academic performance level.

The performance levels are:

- Low
- Average
- High

## Models we plan to compare

- Logistic Regression
- Decision Tree
- Random Forest

Logistic Regression is used as the baseline model.

## Dataset

The data was collected using a Google Form survey.  
At the current stage, we have 111 responses.

The survey includes factors such as:

- attendance
- study hours
- LMS usage
- assignment submission habit
- class participation
- sleep hours
- motivation level
- internet quality
- part-time work
- academic stress
- time management
- study resources

## Evaluation plan

We plan to use:

- accuracy
- precision
- recall
- F1-score
- confusion matrix
- stratified 5-fold cross-validation
